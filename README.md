# day1-rule-based-daily-assistant-agent

## Project overview
This is a beginner Python project that simulates a simple AI agent without using any paid API.

The agent can:
- respond to simple daily-help prompts
- remember the user's last question
- demonstrate agent structure using input, logic, memory, and output

## Why I built this
I am learning AI agents step by step by first understanding the core logic before using advanced tools or paid APIs.

## Features
- Rule-based responses
- Basic memory using a Python dictionary
- Command-line interaction
- Beginner-friendly code structure

## Files
- `day1_agent.py` → main Python code
- `README.md` → project explanation
- `.gitignore` → ignore unnecessary system files
